🖨 LPW

Github

Hello there! LPW is a wrapper around lp, which is a Unix command to send files to a printer (as in a printer that prints paper).

🐧 Linux

LPW was developed for Linux, so you can just run the installation script on the Github page and start using it!

🍎 Mac

If you are on Mac, you may need to run LPW with -CI depending on what terminal emulator you use. Also the installation script won't work, so you will need to manually download the lpw file and execute it.

🪟 Windows

If you are on Windows, sorry but you can't use it! However, there is a GIF on the Github that should sufficiently demonstrate what LPW can do.
